# Apex Workout Plan Generator

**Get your personalized workout plan in seconds!**

This simple web application helps beginners and enthusiasts generate a basic workout plan based on their target muscle groups, available equipment, and fitness goals. No complex algorithms, just straightforward plans to get you moving.

**Features:**
- Select muscle group (Full Body, Upper Body, Lower Body)
- Choose available equipment (Bodyweight, Dumbbells)
- Set your fitness goal (Strength, Endurance)
- Instant plan generation (5-7 exercises, 3 sets of 8-12 reps)
- Printable/PDF output
- Stripe integrated for secure one-time payment ($9.99)

Built with HTML, Tailwind CSS, and vanilla JavaScript for a fast, simple, and effective solution.
